개인정보처리방침

시행일: 2026년 3월 11일

부동산 매수 평가(이하 "앱")은 이용자의 개인정보를 중요시하며, 관련 법령을 준수합니다.

1. 수집하는 개인정보

앱은 개인정보를 수집하지 않습니다.

  • 회원가입이 없으며, 이름·이메일·전화번호 등을 요구하지 않습니다.
  • 매물 평가 데이터는 기기 내부(로컬 저장소)에만 저장됩니다.

2. 공유 기능 이용 시

앱의 실시간 공유 기능을 사용할 경우, 다음 정보가 서버에 임시 저장됩니다:

  • 임의 생성된 사용자 ID (6자리 랜덤 코드)
  • 공유 방의 매물 평가 데이터 (매물명, 점수, 평가 항목)

이 데이터는 개인을 식별할 수 없으며, 공유 종료 시 더 이상 활용되지 않습니다.

3. 제3자 제공

앱은 이용자의 데이터를 제3자에게 제공하지 않습니다.

4. 데이터 보관 및 삭제

  • 로컬 데이터: 앱 삭제 시 자동으로 삭제됩니다.
  • 공유 데이터: 서버에서 주기적으로 정리됩니다.

5. 아동의 개인정보

앱은 만 14세 미만 아동의 개인정보를 수집하지 않습니다.

6. 문의

개인정보 관련 문의는 아래로 연락해 주세요:

이메일: cdn_k@naver.com

7. 변경 사항

본 방침이 변경될 경우, 앱 내 또는 본 페이지를 통해 고지합니다.
